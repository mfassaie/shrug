    - test_error_invalid_output_format — exit non-zero for bad --output value
    - test_error_auth_no_default — auth status without default profile, verify graceful error
    - test_error_messages_contain_hint — generic test that errors include "Hint:" text
    - test_exit_codes_nonzero_on_error — verify errors don't exit 0
